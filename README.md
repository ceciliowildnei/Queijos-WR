# Queijos WR Pedidos

Sistema online simples para controle de pedidos de queijo e leite.

## Stack

- React + Vite
- Supabase como banco de dados online real
- Vercel para publicação

## Variáveis de ambiente

Use no deploy:

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

## Banco de dados

O app usa as tabelas reais:

- `wr_admins`
- `wr_clientes`
- `wr_produtos`
- `wr_pedidos`
- `wr_config`

Não usa `localStorage` como banco principal e não usa `app_state` como armazenamento principal.
